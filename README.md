# 📚 Digital Library Platform

A full-stack digital library application enabling user registration, secure authentication, book browsing, and shopping cart functionality. Built using **Java**, **Spring Boot**, and **React.js**, this project demonstrates end-to-end integration of backend services with a modern front-end interface.

## 👥 Collaboration

- 🎨 **Frontend**: [Sai Niranjan Chitturi](https://github.com/SaiNiranjanChitturi)  
- 🔧 **Backend**: [Anil Kollu](https://github.com/AnilKollu)

## 🚀 Features

- User registration and login with JWT authentication
- Secure role-based access control using Spring Security
- Book listing, details view, and search functionality
- Add/remove books from shopping cart
- RESTful API integration between frontend and backend
- Responsive UI using React.js and Bootstrap
- Real-time UI updates with React Router
- Unit and integration testing with JUnit and Jest

## 🛠️ Tech Stack

- **Frontend**: React.js, React Router, Bootstrap
- **Backend**: Java, Spring Boot, Spring Security, Spring Data JPA
- **Database**: MySQL
- **Authentication**: JWT (JSON Web Token)
- **Build Tools**: Maven
- **Version Control**: Git, GitHub
- **Development Process**: Agile (Scrum)
